# RADIALTRACKER

## PREREQUISITES

1. Matlab (>= 2015a)
2. Matlab Computer Vision System Toolbox

## INSTRUCTIONS

1. Download the folder containing RadialTracker from https://github.com/sjmgarnier/projectRadial/tree/master/matlab/RadialTracker
2. Open Matlab
3. Either:
  a. add the folder containing RadialTracker to the Matlab path,
  b. or set the folder containing RadialTracker as your working directory using the folder browser
4. Type 'RadialTracker' in the Command Window
5. Follow the on-screen directions
